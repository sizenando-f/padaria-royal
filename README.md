# Padaria Royal - Sistema de Controle de Produção

Sistema Full Stack desenvolvido para modernizar o controle de fornadas e qualidade.

## Tecnologias
- **Backend:** NestJS, Prisma, PostgreSQL (Docker)
- **Frontend:** Next.js, Tailwind CSS
- **Infra:** WSL 2, Docker Compose

## Como rodar
1. Suba o banco: `docker compose up -d`
2. Backend: `cd backend && npm run start:dev`
3. Frontend: `cd frontend && npm run dev`
